# Hello World

这是 **zhangpl24** 个人博客的第一篇文章。

站点使用 [Zensical](https://zensical.org/) 从 Markdown 生成静态页面，通过 [GitHub Actions](https://github.com/features/actions) 在推送时自动构建，并发布到 [GitHub Pages](https://pages.github.com/)（用户站点：`https://zhangpl24.github.io/`）。

之后你可以在本仓库的 `docs/blog/` 目录下新增 `.md` 文件，并在 `docs/blog/index.md` 里手动维护链接列表（或使用导航配置）。
